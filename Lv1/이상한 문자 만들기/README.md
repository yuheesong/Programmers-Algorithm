# 배운 점  
## split() vs split(" ")  
- split() : 공백이 1개 혹은 2개 혹은...n개일 경우에도 공백을 1개로 보고 처리한다.  
- split(" ") : 공백이 1개일 경우와 n개일 경우가 다르게 처리된다.  
  
<예시>  
"My name  is   yuyu" 일 때,  
split()을 사용하면 ['My', 'Name', 'is', 'yuyu']  
split(" ")을 사용하면 ['My', 'Name', '', 'is', '', '', 'yuyu']    


## 또한,  
이 문제를 풀면서 중요했던 것은 공백이 2개 이상일때의 처리였다.
